# 💻 Learning Management System Analysis (SQL Project)

## 📌 Project Overview
This project analyzes student enrollment, course performance, and instructor effectiveness using SQL queries on a simulated Learning Management System dataset.

## 🛠 Tools Used
	•	SQLite
	•	SQL Queries
	•	Data Cleaning
	•	Data Transformation
	•	Data Analysis

## 📊 Key Analysis Areas
	•	Student registration trends by month
	•	Average, highest, and lowest scores per course
	•	Instructors whose students perform best
	•	Average number of days to enroll students
	•	Data quality checks: duplicates, nulls, and date formats

## 📈 Key Insights
	•	Most students register in specific months, showing peak enrollment periods.
	•	Certain instructors have higher-performing students on average.
	•	Average enrollment duration helps track administrative efficiency.
	•	Overall student scores vary across courses, highlighting areas for improvement.

## 📂 Dataset
Simulated Learning Management System dataset with the following tables:

	•	Students – StudentID, FirstName, LastName, RegistrationDate
	•	Instructors – InstructorID, InstructorName, Department
	•	Courses – CourseID, CourseName, InstructorID
	•	Enrollments – EnrollmentID, StudentID, CourseID, EnrollmentDate
	•	Assignments – AssignmentID, CourseID, AssignmentName
	•	Grades – GradeID, AssignmentID, StudentID, Score
